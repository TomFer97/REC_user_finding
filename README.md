# REC_user_finding
Tool to find new members of REC

## Esecuzione locale

1. Installare le dipendenze:

```bash
npm install
```

2. Avviare il server:

```bash
npm start
```

3. Aprire il browser su:

```bash
http://localhost:3000
```

Il backend Node/Express espone `/api/query` e proxy le richieste verso i FeatureServer ArcGIS del portale GSE.
